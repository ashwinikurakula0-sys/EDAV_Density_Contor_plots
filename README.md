# 📊 Data Distribution Visualization using Iris Dataset

## 📌 Overview
This  demonstrates Exploratory Data Analysis (EDA) using the Iris dataset. It focuses on understanding data distributions and relationships using Density Plots (KDE) and Contour Plots.

---

## 🎯 Objectives
- Analyze distribution of individual features  
- Compare distributions across species  
- Understand relationships between variables  
- Identify patterns and clusters  

---

## 📂 Dataset
- **Name:** Iris Dataset  
- **Source:** Seaborn library  
- **Features:**
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width  
- **Target:** Species (Setosa, Versicolor, Virginica)

---

## 🛠️ Technologies Used
- Python  
- Seaborn  
- Matplotlib  
- Pandas  

---

## 📈 Visualizations

### 1. Density Plot (KDE)
- Shows distribution of a single variable  
- Example: Sepal Length  

### 2. Multiple Density Plot
- Compares distributions across species  
- Example: Petal Length by Species  

### 3. 2D Filled Contour Plot
- Shows joint distribution of two variables  
- Example: Petal Length vs Petal Width  

### 4. 2D Contour Plot (Lines)
- Displays density levels using contour lines
